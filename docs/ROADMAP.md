# ROADMAP · AMPEL360 (12–18 meses)

## Contexto

Este roadmap define los hitos clave para los dos productos principales del portfolio AMPEL360 en un horizonte de 12 a 18 meses, con foco en incrementar niveles TRL y preparar rutas de certificación.

---

## AMPEL360-T-Air (BWB H₂ Hybrid-Electric)

### Q1 2024 (Meses 1-3)

**QS → FWD (TRL 2-3)**

- [ ] Completar modelos de sizing H₂ (tanques, masa, volumen)
- [ ] Análisis de demanda energética por misión típica
- [ ] Trade studies: configuración BWB vs convencional
- [ ] Selección preliminar de e-fans y arquitectura eléctrica
- [ ] Iniciar diseño conceptual BWB (geometría, integración tanques)

**Entregables**:
- Documento QS/sizing-baseline-v1.0
- Trade study report FWD/bwb-config-study
- Roadmap de proveedores críticos (criogenia, pilas H₂, e-fans)

### Q2 2024 (Meses 4-6)

**FWD → UE (TRL 3-4)**

- [ ] Completar integración criogénica (aislamiento, ventilación, BOG management)
- [ ] Análisis CFD preliminar BWB (drag, lift, stability)
- [ ] Diseño de cabina y distribución de pasajeros (UE)
- [ ] HMI conceptual para crew (energía, H₂ status, emergencias)
- [ ] Safety assessment H₂: fugas, inertización, evacuación

**Entregables**:
- FWD/cryo-integration-report
- CFD results package (baseline drag polar)
- UE/cabin-layout-v1.0
- Preliminary Hazard Analysis (PHA) H₂ systems

### Q3 2024 (Meses 7-9)

**UE → FE (TRL 4-5)**

- [ ] Diseño de arquitectura propulsiva: pila + turbogenerador H₂ + e-fans
- [ ] Integración térmica: cooling loops, waste heat recovery
- [ ] BMS (Battery Management System) conceptual
- [ ] Aviónica: FBW (Fly-By-Wire) para BWB, energy management
- [ ] Subescala BWB (wind tunnel o vuelo RC) para validación aero

**Entregables**:
- FE/propulsion-architecture-v1.0
- FE/thermal-management-study
- Subescala BWB test report (TRL 4)
- Avionics block diagram

### Q4 2024 (Meses 10-12)

**FE → CB (TRL 5)**

- [ ] Iniciar HIL/SIL (Hardware-In-Loop / Software-In-Loop) powertrain
- [ ] Banco de ensayos criogénicos (fill/drain, BOG, thermal cycles)
- [ ] Definición de Special Conditions EASA (H₂ + eléctrico)
- [ ] Matriz de compliance preliminar CS-25 + SC
- [ ] Pre-aplicación EASA para discutir SC y AMC

**Entregables**:
- HIL/SIL test results (e-fan + BMS)
- CB/compliance-matrix-v1.0
- CB/special-conditions-draft
- EASA pre-application package

### Q1 2025 (Meses 13-15)

**CB → QB (TRL 5-6)**

- [ ] Demo e-fan en ground test (thrust, efficiency, noise)
- [ ] Iron-bird eléctrico completo (integración aviónica + propulsión)
- [ ] Taxi tests con propulsión eléctrica (sin vuelo)
- [ ] Actualizar safety cases con datos de ensayos
- [ ] Revisión Design Review (PDR nivel)

**Entregables**:
- QB/e-fan-ground-test-report
- Iron-bird integration report
- Taxi test results
- PDR documentation package

### Q2 2025 (Meses 16-18)

**QB → TRL 6-7 Prep**

- [ ] Preparar prototipo de vuelo (scaled or full-scale)
- [ ] Plan de flight test (fases, KPIs, safety)
- [ ] Finalizar Special Conditions con EASA
- [ ] Funding round para fase de vuelo (EU grants, partners)
- [ ] Iniciar construcción prototipo vuelo

**Entregables**:
- Flight test plan v1.0
- CB/special-conditions-final (agreed with EASA)
- Funding secured para TRL 7+
- Manufacturing plan for prototype

## AMPEL360-T-Space (Suborbital → Orbital)

### Q1 2024 (Meses 1-3)

**QS (Fase A: Suborbital, TRL 2-3)**

- [ ] Definir perfiles de misión suborbital (altitud, duración, payload)
- [ ] Sizing preliminar: masa, propulsión, TPS
- [ ] Trade studies: VTVL vs winged, H₂/O₂ vs other
- [ ] Análisis de aborts y safety (pad, ascent, entry)

**Entregables**:
- QS/mission-profiles-suborbital
- Trade study report (config selection)
- Preliminary abort scenarios

### Q2 2024 (Meses 4-6)

**FWD (TRL 3-4)**

- [ ] Diseño conceptual estructura presurizada
- [ ] Selección preliminar TPS (reusable, cost-effective)
- [ ] GNC (Guidance, Navigation, Control) architecture
- [ ] Integración propulsiva (etapas, reusabilidad)

**Entregables**:
- FWD/structure-concept-v1.0
- TPS materials trade study
- GNC block diagram

### Q3 2024 (Meses 7-9)

**UE → FE (TRL 4)**

- [ ] Diseño de cabina tripulada: 6-10 pax
- [ ] Factores humanos: G-loads, vibración, vistas, confort
- [ ] LSS (Life Support System) conceptual: O₂, CO₂ scrubbing, thermal
- [ ] HMI crew/pax: displays, emergency procedures
- [ ] Drop tests (no propulsion) para validar entry dynamics

**Entregables**:
- UE/cabin-design-space
- LSS preliminary design
- Drop test results (TRL 4)

### Q4 2024 (Meses 10-12)

**CB (Human-Rating prep, TRL 4-5)**

- [ ] Identificar estándares aplicables (ESA, NASA HRP, national)
- [ ] Safety analysis: FMEA, FTA para sistemas críticos
- [ ] Abort system design (pad abort, in-flight abort)
- [ ] Matriz de compliance inicial (ESA/NASA estándares)

**Entregables**:
- CB/standards-matrix-space
- Abort system conceptual design
- Safety analysis reports

### Q1 2025 (Meses 13-15)

**QB (TRL 5)**

- [ ] Hover tests (vertical takeoff, hover, landing)
- [ ] Captive carry tests (aerodynamic validation)
- [ ] Propulsion ground tests (full thrust, throttle)
- [ ] LSS functional tests (closed loop)

**Entregables**:
- Hover test report
- Captive carry results
- Propulsion test data
- LSS test results

### Q2 2025 (Meses 16-18)

**QB → TRL 6 Prep**

- [ ] Preparar vuelos libres suborbitales (no tripulados)
- [ ] Finalizar abort system y demostración
- [ ] Pre-certificación con autoridades (ESA, national)
- [ ] Funding para fase tripulada (Fase B: LEO)

**Entregables**:
- Free flight test plan
- Abort system demo report
- Pre-certification discussions summary
- Funding secured for crewed flights
